Instructions:
NopCommerce (http://www.nopcommerce.com/) is an open source e-commerce onlie store platform. This application is special for analyzing order data from 
a NopCommerce online store.
1.Build the application in your localhost.
2.Open the index.html page.
3.Get the sample data from sampleData folder or download from the home page.
4.After upload the file, please make sure hit the save button to use Charts and Maps page.
5.Make sure clear data and refresh page before each new uploading.
